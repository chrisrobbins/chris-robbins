This is a pet project That I am developing with two UX designers from Fender and IBM. The idea started in October 2016. Since then We have had weekly meetings where I facilitated the use of React Native in order to develop for IOS and Android with the most simplicity. We plan to launch Nov. 1! Please check out the app on my github.

Lyricist

Lyricist is an app that allows you to search for your favorite song and return both the lyrics and the video in case you want to sing along or learn how to play along with your instrument of choice. I worked with two other developers on this project. We designed and coded our mvp in five days. We had to make our own lyrics api because other lyrics api's are very expensive. Try doing a search for "Knockin on Heaven's Door"

Technologies:   React   |   |   CSS   |   Node   |   Express

Stratfor Charts

Stratfor hired me to do some data vis for them. I researched some different libraries and decided on Highcharts. Highcharts has a lot of neat stuff right out of the box, but mixing it into html can be a bit of a challenge! One I gladly accepted. This chart is highly interactive and ultimately customized.

Paplove

Paplove is an online store idea inspired by a UX software designer. She gave me a Sketch wireframe of the homepage and explained to me how she wanted it to act when shrinking the window. It's a demonstration of UI layout and responsive development using media queries, therfore, nothing else works on the site. I got all of my assets and dimensions from the sketch wireframe and coded this homepage in HTML, CSS, and JQUERY.

Technologies:   HTML   |   CSS   | JavaScript   |   jQuery
